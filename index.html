Sobre Mí

Soy un Desarrollador de Software apasionado por la innovación y la creación de productos digitales que generan un impacto real. Soy orgullosamente graduado de la Universidad Tecnológica de Santa Catarina, donde adquirí bases sólidas en ingeniería de software y resolución de problemas complejos.

Mi enfoque principal está en la intersección entre el Desarrollo Web moderno y la Inteligencia Artificial. Me especializo en construir aplicaciones web que no solo se ven bien y son rápidas, sino que también integran modelos de IA para automatizar tareas, predecir comportamientos y ofrecer experiencias de usuario hiper-personalizadas.

Proyectos Destacados (Web & IA)

Una selección de aplicaciones web impulsadas por Inteligencia Artificial.

NeuroChat CRM

Sistema CRM web que utiliza Procesamiento de Lenguaje Natural (NLP) para analizar el sentimiento de los correos y mensajes de los clientes, priorizando automáticamente los tickets urgentes o clientes insatisfechos.

React Node.js OpenAI API MongoDB

Visionary E-commerce

Plataforma de comercio electrónico con un motor de recomendación visual. Los usuarios suben una foto de una prenda y la IA (Visión Computacional) busca productos similares en el inventario de la tienda en milisegundos.

Next.js Python TensorFlow AWS S3

DocuMind Legal Assistant

Herramienta web diseñada para despachos de abogados. Los usuarios suben contratos largos en PDF y un LLM entrenado extrae las cláusulas de riesgo, fechas de vencimiento y genera un resumen ejecutivo de 1 página.

Vue.js FastAPI LangChain Pinecone

FinPredict Dashboard

Dashboard financiero interactivo que procesa datos históricos de ventas de PyMEs y utiliza modelos de series temporales (Prophet) para predecir el flujo de caja de los próximos 3 meses, visualizado con gráficos dinámicos.

Svelte Python Pandas Chart.js

AutoAgent Support Web

Un widget de soporte técnico incrustable para cualquier sitio web. Implementa agentes autónomos basados en la base de conocimientos de la empresa que pueden resolver problemas técnicos paso a paso sin intervención humana.

Vanilla JS Express RAG Architecture PostgreSQL
